# Documentation

The documents are written in markdown format. 


## Install Mkdocs

```
pip install Mkdocs
```

## Preview

In the current directory
```
mkdocs serve
```

Open up <http://127.0.0.1:8000/>

## Add New Pages

* Create new markdown page under `docs`
* Update configuration file `mkdocs.yml`
